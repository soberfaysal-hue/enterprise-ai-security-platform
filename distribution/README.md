# Enterprise AI Security Red Teaming Platform

## Quick Start

1. **Install Python 3.8+** if not already installed

2. **Run the platform:**
   ```
   double-click RUN.bat
   ```

3. **Open in browser:**
   ```
   http://localhost:8080
   ```

## Configuration

Edit `backend/.env` to add your API keys:
- `GOOGLE_API_KEY=your-google-key` (already configured)
- `OPENAI_API_KEY=your-openai-key`
- `ANTHROPIC_API_KEY=your-anthropic-key`

## Features

- Security vulnerability testing for enterprise AI (ChatGPT, Claude, Gemini)
- Attack technique simulation (poetry, narrative, metaphor, etc.)
- Risk scoring and compliance mapping (SOC2, ISO27001, GDPR, etc.)
- Export reports (CSV, PDF)
- Detailed results with mitigation recommendations
